# Door Control Summar
- Commands to open a door of your choice.
- The commands are keybindable through FiveM settings.

# Events

| Event                             | Description                                  | Paramters(s)           |
| --------------------------------- | -------------------------------------------- | ---------------------- |
| TR:Client:DoorControl:OpenDoor    | Open a vehicles door by the VehicleSeat enum | enum                   |
